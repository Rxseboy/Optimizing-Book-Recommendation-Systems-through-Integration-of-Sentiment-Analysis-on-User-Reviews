The Intelligence Pipeline

Five stages transform raw Goodreads reviews into sentiment-enhanced recommendations

01

Data Loading & Cleaning

Language detection with langdetect, word-count filter, deduplication. English-only reviews of Carissa Broadbent books are retained.

18,984 → 12,439 reviews
02

NLP Preprocessing

Lowercase → contraction expansion → noise removal → tokenization → stopword removal → multi-pass lemmatization via NLTK.

Clean text + nouns extracted
03

Sentiment Analysis

KNN classifier on CountVectorizer features · SMOTE oversampling · optimal K search · predicts sentiment for 'No Rating' reviews.

91.42% Accuracy · F1 0.91
04

Collaborative Filtering

User–user cosine similarity matrix · top-20 neighbour selection · mean-centred predicted ratings. Filters active users (≥5 interactions).

RMSE 0.68 · MAE 0.42
05

Hybrid Ranking

Final score: 0.85 × predicted_rating + 0.15 × sentiment_score. Items already seen by the user are excluded, fallback to global top books.
